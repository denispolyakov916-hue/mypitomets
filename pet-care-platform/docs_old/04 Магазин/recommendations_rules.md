# Логика рекомендаций по подкатегориям

Источник категорий: `docs/04 Магазин/Структура категорий/category_tree.md`.
Для каждой подкатегории указаны 4 наиболее совместимые подкатегории
(рекомендательные связи между категориями).

Принципы:
- По возможности давать 1–2 доп. подкатегории из других разделов.
- Для корма: миски/поилки, лакомства, витамины.
- Для туалетов: наполнители, совочки, пакеты, салфетки.
- Для ухода: щетки/расчески, салфетки, шампуни/кондиционеры, полотенца.
- Для амуниции: поводки/ошейники, адресники, аксессуары.
