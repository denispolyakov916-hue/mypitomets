# Контекст реализации системы курсов обучения питомцев
## Питомец+ Platform

### 🎯 Цель реализации
Реализовать полную систему обучения для владельцев домашних животных через различные форматы курсов с персонализацией по PetID.

### 📋 Основные требования из FREE_COURSES_CONTENT.md

#### Сущности системы:
- **Пользователь** (владелец животного)
- **Питомец** (профиль животного с расширенным PetID)
- **Курс** (общий контейнер)
- **Урок** (элемент курса)
- **Типы уроков**: Видео, Текст, Интерактивный, Смешанный, Вебинар, Мастер-класс
- **Прогресс прохождения**
- **Комментарии** (к урокам и курсам)
- **Оценки курсов**

#### Архитектура:
1. **Бекенд**: Django REST Framework
2. **Фронтенд**: React с Zustand
3. **База данных**: PostgreSQL
4. **Персонализация**: Через расширенный PetID

### 🗄️ Модели данных (расширенные)

#### PetID (расширенная модель)
```python
# Базовые поля (обязательные для персонализации)
- behavior_type: str  # тип поведения (спокойный, активный, агрессивный, трусливый)
- activity_level: str  # уровень активности (низкий, средний, высокий)
- social_level: str   # социализация (домашний, уличный, социальный)

# Расширенные поля (опциональные)
- training_experience: str  # опыт дрессировки
- health_issues: list       # проблемы здоровья (JSON)
- special_needs: list       # особые потребности (JSON)
- preferred_activities: list # предпочитаемые активности (JSON)
- behavioral_problems: list  # поведенческие проблемы (JSON)
```

#### Course (расширенная модель)
```python
- title, description, image
- format_type: str  # video/text/interactive/mixed/webinar/masterclass
- level: str        # beginner/intermediate/advanced
- duration: int     # в минутах
- instructor: str
- pet_types: list   # ['dog', 'cat', 'universal']
- category: str     # основы/питание/здоровье/уход/поведение/дрессировка
- lessons_count: int
- is_free: bool
- lessons: FK to Lesson (ManyToMany через CourseLesson)

# ===== ПЕРСОНАЛИЗАЦИЯ ПО ХАРАКТЕРИСТИКАМ ПИТОМЦА =====
- recommended_behavior_types: list    # рекомендуемые типы поведения
- recommended_activity_levels: list   # уровни активности
- recommended_social_levels: list     # уровни социализации
- min_training_experience: str        # минимальный опыт дрессировки
- compatible_health_issues: list      # совместимые проблемы здоровья
- addresses_special_needs: list       # решаемые особые потребности
- suitable_activities: list           # подходящие активности
- addresses_behavioral_problems: list # решаемые поведенческие проблемы
```

#### Новые модели
```python
# Lesson - урок курса
- course: FK to Course
- title: str
- content_type: str  # video/text/interactive/mixed/webinar/masterclass
- content: JSONField  # структура зависит от типа
- duration: int
- order: int          # порядок в курсе
- is_required: bool

# UserCourseProgress - прогресс пользователя по курсу
- user: FK to User
- course: FK to Course
- pet: FK to Pet       # привязка к конкретному питомцу
- status: str          # not_started/in_progress/completed
- progress_percent: int
- started_at: datetime
- completed_at: datetime

# UserLessonProgress - прогресс по уроку
- user_progress: FK to UserCourseProgress
- lesson: FK to Lesson
- status: str          # not_started/viewed/completed
- time_spent: int      # время просмотра в секундах
- attempts_count: int  # для интерактивных уроков
- completed_at: datetime

# Comment - комментарии к урокам/курсам
- user: FK to User
- lesson: FK to Lesson (nullable)
- course: FK to Course (nullable)
- content: str
- created_at: datetime
- updated_at: datetime
- parent: FK to Comment (для ответов)
- attachments: JSONField (изображения/файлы)
- is_approved: bool (модерация)

# CommentLike - лайки комментариев
- user: FK to User
- comment: FK to Comment
- is_like: bool (лайк/дизлайк)
- created_at: datetime

# Rating - оценки курсов
- user: FK to User
- course: FK to Course
- pet: FK to Pet (оценка для конкретного питомца)
- rating: int (1-5)
- review: str (отзыв)
- created_at: datetime
- updated_at: datetime
- is_approved: bool (модерация)
```

### 🔗 API Endpoints (расширения)

#### Курсы
- `GET /api/courses/` - список с фильтрами (pet_type, format, level, category)
- `GET /api/courses/{id}/` - детали курса
- `POST /api/courses/{id}/enroll/` - запись на курс
- `GET /api/courses/{id}/progress/` - прогресс по курсу
- `PUT /api/courses/{id}/progress/` - обновление прогресса

#### Уроки
- `GET /api/courses/{course_id}/lessons/` - уроки курса
- `GET /api/lessons/{id}/` - детали урока
- `POST /api/lessons/{id}/complete/` - завершение урока
- `PUT /api/lessons/{id}/progress/` - обновление прогресса урока

#### Комментарии
- `GET /api/lessons/{id}/comments/` - комментарии к уроку
- `POST /api/lessons/{id}/comments/` - добавить комментарий
- `GET /api/courses/{id}/comments/` - комментарии к курсу
- `PUT /api/comments/{id}/` - редактировать комментарий
- `DELETE /api/comments/{id}/` - удалить комментарий
- `POST /api/comments/{id}/like/` - лайкнуть комментарий
- `POST /api/comments/{id}/dislike/` - дизлайкнуть комментарий

#### Оценки
- `GET /api/courses/{id}/ratings/` - оценки курса
- `POST /api/courses/{id}/ratings/` - поставить оценку
- `PUT /api/ratings/{id}/` - редактировать оценку
- `DELETE /api/ratings/{id}/` - удалить оценку

### 🎨 Фронтенд компоненты

#### Типы курсов:
1. **VideoCourse** - видеоплеер с таймлайном
2. **TextCourse** - читалка с интерактивными элементами
3. **InteractiveCourse** - пошаговые задания
4. **MixedCourse** - комбинированный интерфейс
5. **Webinar** - видео + чат
6. **Masterclass** - мультимедиа презентация

#### Ключевые компоненты:
- `LessonPlayer` - универсальный плеер уроков
- `ProgressTracker` - отслеживание прогресса
- `CommentsSection` - комментарии к урокам
- `RatingWidget` - оценки курсов
- `PetSelector` - выбор питомца для курса
- `CourseRecommendations` - персонализированные рекомендации
- `CoursePersonalizationWidget` - информация о совместимости курса с питомцем
- `useProgress` - хук управления прогрессом
- `useLessonTimer` - хук отслеживания времени уроков

### 🔄 Логика прохождения курсов

#### Начало курса:
1. Пользователь выбирает курс
2. Система предлагает выбрать питомца
3. Создается UserCourseProgress
4. Начинается первый урок

#### Во время прохождения:
- Прогресс сохраняется автоматически каждые 30 сек
- Синхронизация между устройствами
- Привязка к конкретному питомцу

#### Завершение:
- Сертификат для пользователя и питомца
- Запрос оценки и отзыва
- Рекомендация следующего курса

### 🎯 Персонализация

#### Фильтры по PetID:
- **Тип животного**: dog/cat/universal
- **Поведение**: спокойный/активный/агрессивный/трусливый
- **Активность**: низкая/средняя/высокая
- **Социализация**: домашний/уличный/социальный
- **Опыт дрессировки**: нет/базовый/средний/продвинутый/профессиональный
- **Здоровье**: проблемы и ограничения
- **Особые потребности**: индивидуальные требования
- **Предпочитаемые активности**: виды занятий
- **Поведенческие проблемы**: агрессия, страх, гиперактивность и др.

#### Интеллектуальные рекомендации:
- **Совместимость курса с питомцем** (балл 0-100)
- **Персонализированные советы** по прохождению
- **Предупреждения** о противопоказаниях
- **Курсы для решения проблем** питомца
- **Рекомендации по активностям** согласно предпочтениям
- **Адаптация времени обучения** под характеристики питомца
