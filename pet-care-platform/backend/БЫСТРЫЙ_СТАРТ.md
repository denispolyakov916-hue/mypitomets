# ⚡ Быстрый старт

## 📋 Порядок действий

### 🔄 После обновления кода с Linux

```bash
cd D:\pet_develop\Pet_dev\pet-care-platform\backend
run_backend.bat
```

### 🆕 Первая настройка проекта

```bash
# 1. Сначала настройте базу данных (запустите один раз)
cd D:\pet_develop\Pet_dev\pet-care-platform\backend
setup_database.bat

# 2. Затем запускайте бэкенд
run_backend.bat
```

## 📝 Что делают скрипты

### `setup_database.bat`
- ✅ Проверяет PostgreSQL
- ✅ Создает пользователя БД `pitomets`
- ✅ Создает базу данных `pitomets_db`
- ✅ Настраивает права доступа
- ✅ Создает файл `.env` с настройками

### `run_backend.bat`
- ✅ Проверяет Python
- ✅ Создает виртуальное окружение
- ✅ Устанавливает зависимости
- ✅ Проверяет подключение к БД
- ✅ Применяет миграции Django
- ✅ Запускает сервер на http://localhost:8000

## 🔑 Пароль от БД

**Пароль от пользователя postgres PostgreSQL** нужно ввести только при первом запуске `setup_database.bat`

## 🌐 Доступ после запуска

- **API**: http://localhost:8000/api/
- **Админка**: http://localhost:8000/admin/

## ❗ Проблемы?

Если что-то не работает:
1. Убедитесь что PostgreSQL запущен
2. Проверьте настройки в файле `.env`
3. Попробуйте запустить скрипты по отдельности
