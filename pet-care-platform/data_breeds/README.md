## 📁 Описание файлов

### 1. `breeds.json` - Основная таблица пород

**Что хранит:** Базовые характеристики каждой породы

**Ключевые поля:**
- `id` - Уникальный идентификатор
- `species` - Вид (dog/cat)
- `name`, `name_en`, `slug` - Названия
- **Размеры:** `size_category`, `weight_min/max`, `lifespan_min/max`
- **Поведение:** `energy_level`, `trainability`, `intelligence`, `friendliness_*`
- **Уход:** `grooming_frequency`, `shedding_level`, `coat_type`
- **Здоровье:** `health_risk_level`, `hypoallergenic`, `brachycephalic`

**Используется в:**
- ✅ PetID - автозаполнение и сравнение с эталоном
- ✅ Магазин - подбор корма и аксессуаров по размеру
- ✅ Курсы - рекомендации по обучаемости
- ✅ Дневник - генерация напоминаний по уходу

**Пример:**
```json
{
  "id": 1,
  "species": "cat",
  "name": "Британская Короткошерстная",
  "size_category": "medium",
  "weight_min": 4.0,
  "weight_max": 8.0,
  "energy_level": "medium",
  "health_risk_level": "medium"
}
```

---

### 2. `breed_health.json` - Генетические риски

**Что хранит:** ТОЛЬКО породоспецифичные генетические заболевания

**Исключено:** Паразиты, ожирение, общие инфекции (не породоспецифичны)

**Ключевые поля:**
- `breed_id` - Связь с breeds.json
- `condition_name` - Название заболевания
- `condition_type` - Тип (genetic/congenital)
- `affected_system` - Затронутая система организма
- `severity` - Тяжесть (low/medium/high)
- `prevalence_percent` - Распространенность в породе (%)
- `prevention` - Профилактика
- `screening` - Рекомендуемые обследования

**Используется в:**
- ✅ PetID - предупреждения о рисках породы
- ✅ Магазин - рекомендации кормов для профилактики
- ✅ Дневник - напоминания о скрининговых обследованиях

**Пример:**
```json
{
  "breed_id": 1,
  "condition_name": "Дисплазия тазобедренных суставов",
  "affected_system": "musculoskeletal",
  "severity": "high",
  "prevalence_percent": 15.0,
  "prevention": "Контроль веса, добавки с глюкозамином"
}
```

---

### 3. `breed_nutrition.json` - Рекомендации по питанию

**Что хранит:** Специфичные для породы потребности в питании

**Ключевые поля:**
- `breed_id` - Связь с breeds.json
- `protein_need` - Потребность в белке (low/medium/high/very_high)
- `calorie_density` - Калорийность корма (low/medium/high)
- `diet_type` - Тип корма (dry/wet/mixed)
- `feeding_frequency` - Частота кормлений
- `special_considerations` - Особые рекомендации
- `common_allergens` - Породоспецифичные аллергены (если есть)

**Используется в:**
- ✅ Подбор корма - расчет БЖУ и калорий
- ✅ Магазин - фильтрация по типу корма
- ✅ PetID - рекомендации по кормлению

**Пример:**
```json
{
  "breed_id": 3,
  "protein_need": "medium",
  "calorie_density": "low",
  "diet_type": "wet",
  "special_considerations": "Брахицефалы - легкоусвояемая пища, мелкие кусочки",
  "common_allergens": ["курица"]
}
```

---

### 4. `breed_care.json` - Специфичный уход

**Что хранит:** ТОЛЬКО породоспецифичные процедуры ухода

**Общие процедуры** (чистка зубов, когтей, лотка) → в `common_data.json`

**Ключевые поля:**
- `breed_id` - Связь с breeds.json
- `care_category` - Категория (coat/skin/ears/respiratory/eyes)
- `procedure` - Название процедуры
- `frequency` - Частота выполнения
- `importance` - Важность (low/medium/high/critical)
- `season` - Сезонность (all/spring/summer/autumn/winter)
- `notes` - Примечания

**Используется в:**
- ✅ Дневник здоровья - персонализированные напоминания
- ✅ Магазин - рекомендации товаров для ухода
- ✅ PetID - предупреждения об особом уходе

**Примеры:**
- Французский бульдог → Ежедневная чистка складок кожи (critical!)
- Мейн-кун → Ежедневное расчесывание длинной шерсти
- Сфинкс → Купание раз в 1-2 недели (из-за выделения жира)
- Шотландская вислоухая → Еженедельная чистка ушей

---

### 5. `common_data.json` - Общие данные ⭐

**Что хранит:** Данные, одинаковые для ВСЕХ собак/кошек

**Разделы:**

#### Вакцинации (`vaccinations`)
- **Собаки:** DHPP (обязательная), Бешенство (обязательная), Бордетелла, Лептоспироз
- **Кошки:** FVRCP (обязательная), Бешенство (обязательная), FeLV

#### Процедуры ухода (`care_procedures`)
- **Собаки:** Чистка зубов, стрижка когтей, чистка ушей, купание
- **Кошки:** Чистка зубов, стрижка когтей, чистка лотка, расчесывание

#### Профилактика паразитов (`parasite_prevention`)
- Обработка от блох и клещей (ежемесячно)
- Дегельминтизация (раз в 3 месяца)

**Используется в:**
- ✅ Дневник здоровья - автогенерация стандартных напоминаний для всех питомцев
- ✅ PetID - базовые рекомендации по уходу

## 💡 Как использовать

### Для разработчиков:

```python
# Загрузка данных породы
import json

with open('data_breeds/breeds.json', 'r', encoding='utf-8') as f:
    breeds = json.load(f)

# Найти породу по ID
breed = next(b for b in breeds if b['id'] == 1)
print(breed['name'])  # "Британская Короткошерстная"

# Получить риски здоровья
with open('data_breeds/breed_health.json', 'r', encoding='utf-8') as f:
    health = json.load(f)

breed_risks = [h for h in health if h['breed_id'] == breed['id']]
for risk in breed_risks:
    print(f"- {risk['condition_name']} ({risk['severity']})")
```
