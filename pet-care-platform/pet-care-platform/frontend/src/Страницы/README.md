# Страницы (для редизайна)

Папка `src/Страницы/` содержит “копии-обёртки” существующих страниц фронтенда с **русскими названиями файлов**.

## Как открыть

- Откройте в браузере: `/pages`
- Там есть оглавление со ссылками на все страницы.

## Важно

- Часть страниц использует данные с API и требует JWT. Без авторизации/данных вы можете увидеть пустые состояния или ошибки — это нормально для задачи редизайна.
- Страницы с параметрами имеют примеры (например `id=1`, `slug=labrador-retriever`) — при необходимости поменяйте значения в URL.
