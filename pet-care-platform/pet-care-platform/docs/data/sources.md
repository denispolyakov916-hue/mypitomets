# Источники данных

## Назначение

Этот документ описывает, какие источники данных допустимы для справочников Питомец+ и как подтверждать происхождение записей.

Главное правило: данные, влияющие на рекомендации, здоровье, питание, породы или уход, должны иметь проверяемый источник. Если источник не указан, запись считается неподтвержденной.

## Типы данных проекта

| Тип данных | Примеры | Требование к источникам |
|------------|---------|-------------------------|
| Породы | Название, классификация, стандарт, страна происхождения | Официальный реестр или породный стандарт. |
| Вес и размер | Минимальный/максимальный вес, размерная категория | Официальный стандарт, ветеринарный справочник или подтвержденная методология. |
| Здоровье | Риски, предрасположенности, аллергии | Ветеринарный источник или программа породного здоровья. |
| Питание | Калорийность, коэффициенты, правила подбора | Ветеринарные рекомендации, WSAVA/AAHA/AAFP или внутренняя методология с ссылками. |
| Товары | Категории, характеристики, SKU | Поставщик, фид, ручная модерация или каталог. |
| Курсы | Темы, уроки, материалы | Автор курса, редактор, экспертная проверка. |

## Уровни доверия

| Уровень | Значение |
|---------|----------|
| `high` | Данные подтверждены официальным реестром, стандартом, ветеринарной организацией или первичным источником. |
| `medium` | Данные подтверждены надежным вторичным источником, но требуют периодической ручной сверки. |
| `low` | Источник неофициальный, неполный или запись внесена вручную без достаточного подтверждения. |
| `unverified` | Источник отсутствует или не проверен. Такие записи нельзя использовать для сильных утверждений. |

## Обязательные поля источника

Для справочников пород, здоровья, питания и ухода каждая запись должна иметь:

| Поле | Описание |
|------|----------|
| `source_organization` | Организация или владелец источника. |
| `source_title` | Название страницы, стандарта, документа или набора данных. |
| `source_url` | Прямая ссылка на источник. |
| `source_accessed_at` | Дата последней проверки источника. |
| `source_type` | `registry`, `standard`, `guideline`, `vendor_feed`, `manual_review`, `internal_methodology`. |
| `data_confidence` | `high`, `medium`, `low`, `unverified`. |
| `notes` | Ограничения, спорные моменты, различия между реестрами. |

## Источники пород собак

| Источник | Назначение | URL | Статус |
|----------|------------|-----|--------|
| Federation Cynologique Internationale, FCI | Международная классификация пород, группы, секции, номера стандартов. | https://www.fci.be/en/nomenclature/ | Основной источник для собак. |
| American Kennel Club, AKC | Сверка англоязычных названий, групп, популярных пород и описаний. | https://www.akc.org/dog-breeds/ | Дополнительный источник. |
| The Royal Kennel Club | Сверка названий и породных групп. | https://www.royalkennelclub.com/search/breeds-a-to-z/ | Дополнительный источник. |
| OFA CHIC | Породные программы медицинских проверок собак. | https://ofa.org/chic-programs/browse-by-breed/ | Источник для health-рекомендаций, не для названий пород. |

Правило: если FCI, AKC и The Royal Kennel Club по-разному классифицируют породу, это фиксируется в `notes`. Нельзя автоматически объединять породы или разновидности без явного решения.

## Источники пород кошек

У кошек нет единого универсального мирового списка, который одинаково признают все организации. Поэтому база должна хранить не только породу, но и реестр, которым она подтверждена.

| Источник | Назначение | URL | Статус |
|----------|------------|-----|--------|
| Federation Internationale Feline, FIFe | Породные стандарты и EMS-коды. | https://fifeweb.org/cats/breeds/ | Основной источник для кошек, если порода есть в FIFe. |
| The International Cat Association, TICA | Сверка признанных пород и стандартов TICA. | https://tica.org/ | Дополнительный источник, прямые страницы пород нужно проверять вручную. |
| World Cat Federation, WCF | Сверка пород и стандартов WCF. | https://wcf.de/en/ | Дополнительный источник. |
| Cat Fanciers' Association, CFA | Сверка пород и стандартов CFA. | https://cfa.org/breeds/ | Дополнительный источник. |

Правило: не писать "все породы кошек". Корректная формулировка: "справочник пород кошек, сверяемый с несколькими фелинологическими реестрами".

## Источники питания и здоровья

| Источник | Назначение | URL | Статус |
|----------|------------|-----|--------|
| WSAVA Global Nutrition Guidelines | Общие рекомендации по оценке питания и рациона. | https://wsava.org/global-guidelines/global-nutrition-guidelines/ | Основной источник для принципов питания. |
| AAHA/AAFP Feline Life Stage Guidelines | Возрастные рекомендации для кошек. | https://www.aaha.org/resources/2021-aaha-aafp-feline-life-stage-guidelines/ | Источник для life stage логики кошек. |
| OFA CHIC | Породные проверки здоровья собак. | https://ofa.org/chic-programs/browse-by-breed/ | Источник для породных health-risk подсказок. |

Правило: медицинские формулировки должны быть осторожными. Использовать "риск", "предрасположенность", "рекомендуется обсудить с ветеринаром", но не "диагноз" и не "лечение".

## Источники товаров

Товарные данные могут приходить из:

- поставщика;
- ручной загрузки каталога;
- внутреннего справочника категорий;
- фида интеграции.

Для товарных данных нужно хранить:

- источник загрузки;
- дату импорта;
- дату последней проверки;
- статус модерации;
- связь с категорией;
- признак доступности.

## Запрещенные практики

- Копировать большие описания с сайтов без права использования.
- Смешивать официальные стандарты и маркетинговые описания без указания источника.
- Использовать неподтвержденные health-риски в персональных рекомендациях.
- Считать отсутствие породы в одном реестре ошибкой, если она признана другим реестром.
- Обещать "полное покрытие" без отчета покрытия.

## Минимальный процесс обновления справочника

1. Добавить или изменить запись.
2. Указать источник и дату проверки.
3. Проверить дубликаты по `species + slug`.
4. Проверить статус признания.
5. Обновить отчет покрытия.
6. Прогнать импорт и тесты.
7. Обновить документацию, если изменилась схема или методология.
