# Frontend Design Reference — Питомец Плюс (эталон pitometsplus.ru)

> Источник истины бренда — статический лендинг `frontend/public/landing/index.html`
> (это та же сборка, что развёрнута на pitometsplus.ru). Документ фиксирует эталонный
> визуальный язык и расхождения текущего приложения. Это справочник, не код.
> Статус: Этап 1 (аудит). Дата: см. git.

## 1. Цветовая палитра (строго 5 цветов + служебные)
| Токен | HEX | Роль |
|---|---|---|
| primary (deep purple) | `#522f81` | основной бренд: заголовки, текст, кнопки, иконки |
| violet | `#C86BFA` | акцентный фиолетовый: ссылки, hover, кольца фокуса |
| gold / orange | `#fbba2d` | **CTA** и тёплый акцент |
| yellow-light | `#F0EB93` | мягкий жёлтый фон/акцент |
| white / milk | `#F5F5F5` | фон страниц (молочный, НЕ чисто белый) |
| card | `#FFFFFF` | поверхность карточек |
| текст на фоне | `#522f81` | основной текст — фиолетовый, а не серый/чёрный |

Служебные: success `#22c55e`, warning `#fbba2d`, danger `#EF4444`, info `#C86BFA`.
Правило: **серый текст (#666/#1F2937/gray-900) для заголовков и body — НЕ бренд**; использовать `#522f81`.

## 2. Типографика
- **Заголовки:** Manrope 600–800.
- **Body:** на лендинге — Manrope; в приложении — Inter (допустимо, но заголовки строго Manrope).
- Размеры заголовков (лендинг): h1 `clamp(1.75rem … 3.75rem)` (`text-5xl/6xl`), h2 `text-4xl/5xl`, bold.
- Цвет заголовков: `#522f81` или градиент (`heading-gradient`); на тёмно-фиолетовых секциях — белый.
- Body line-height 1.5–1.6.

## 3. Скругления (radius)
| Элемент | Эталон | Сейчас в приложении |
|---|---|---|
| Кнопки/CTA | **pill `9999px`** (`rounded-full`) | `rounded-xl` (12px) ❌ |
| Карточки крупные | `1.5rem` (24px) | `rounded-2xl` (16px) 🟡 |
| Карточки/блоки мелкие | `1rem`–`1.25rem` (16–20px) | разнобой 🟡 |
| Инпуты | 14–16px (мягко) | auth 14px ✓, прочие разные 🟡 |
| Круги (маскот, иконки) | `50%` | ок |

## 4. Тени (brand-tinted, не нейтральные)
- **Карточки:** мягкие, фиолетовые: `0 4px 20px rgba(82,47,129,0.06)`, hover `0 16px 40px rgba(82,47,129,0.12)` + `translateY(-6px) scale(1.02)`.
- **Крупные блоки:** `0 20px 48px rgba(82,47,129,0.14)`, `0 10px 40px rgba(82,47,129,0.12)`.
- **CTA (золото):** свечение `0 6px 24px rgba(251,186,45,0.45), 0 2px 8px rgba(251,186,45,0.25)`, hover усиление.
- ❌ Сейчас в приложении: нейтральные `shadow-sm/md` (серые) — не бренд.

## 5. Кнопки
- **Primary / CTA:** фон `#fbba2d` (gold), текст `#522f81`, font-weight 600–700, **pill**, padding `10px 20px` (компакт) … `px-8 py-4` (герой), **золотое свечение**, hover `brightness(1.1)` + `scale(1.02–1.05)`. На финальном CTA — пульсация.
- **Secondary / login:** текстовая/контурная, фиолетовая; hover — насыщеннее.
- **На тёмно-фиолетовом фоне:** контурная белая кнопка.
- ❌ Расхождение: в `ui/Button.jsx` primary = сплошной фиолетовый (`bg-primary-600`) с серой тенью и `rounded-xl`; «золотой» — отдельный вариант `accent`. Эталон: главный CTA = золотая pill со свечением.

## 6. Карточки
- Поверхность белая на молочном фоне, radius **24px**, фиолетовая мягкая тень, hover-подъём.
- Медиа-карточки: фото + фиолетовый оверлей `rgba(82,47,129,0.4)`, белый текст снизу.
- min-height ~200–240px, контент с padding `1.25rem`.
- 🟡 Сейчас `ui/Card.jsx`: radius 16px, серые тени, слабый hover.

## 7. Секции
- Вертикальный ритм `clamp(2rem, 6vh, 4rem)`, гориз. `clamp(1rem, 4vw, 2rem)`.
- Контейнер `max-w-7xl` по центру.
- Чередование: молочный фон / белые карточки / финальный CTA на тёмно-фиолетовом.
- Заголовок секции: крупный Manrope bold, primary/gradient; подзаголовок — приглушённый фиолетовый.
- Структура лендинга: `#hero` → `#domains-section` → `#products` → `#features` → `#how-it-works` → `#cta-final`.

## 8. Формы
- Инпуты: мягкий лавандовый фон (`rgba(82,47,129,0.1)`), radius 14px, фокус — золотая рамка `#fbba2d` + кольцо `rgba(251,186,45,0.35)`, иконка справа фиолетовая.
- Лейблы видимые (не только placeholder), ошибки под полем, helper-текст.
- Кнопка submit — брендовый градиент `#522f81 → #fbba2d` или золото.
- ✓ Auth уже близко к эталону (после polish). 🟡 Прочие формы — разнобой (серые рамки, gray-200).

## 9. Модальные окна
- Контейнер radius ~30px, мягкая фиолетовая тень `0 0 40px rgba(82,47,129,0.3)`, лёгкий backdrop-blur.
- Scrim фон 40–60% затемнения, центрирование, закрытие по клику вне/ESC.
- Заголовок Manrope, акценты бренда.
- 🟡 Сейчас единого `BrandModal` нет; модалки разрозненные.

## 10. Мобильные правила (360–414px)
- Mobile-first; брейкпоинты 375/768/1024/1440.
- Нижняя навигация: ≤5 пунктов (сейчас 6 — пересмотреть), иконки+подпись, активный выделен.
- Touch-таргеты ≥44px, отступы ≥8px, safe-area (notch / home-indicator).
- Без горизонтального скролла; контент стекается (auth: gold-панель → форма).
- Кнопки-pill компактные (`padding: 0.45rem 1.1rem`).

## 11. Маскот / иллюстрации
- Маскот «Пуф» (фиолетовый монстр с золотыми рожками) — герой и плавающий виджет поддержки (низ-право), круг с золотой обводкой.
- Иллюстрации тёплые, дружелюбные, на молочном фоне; не перегружать.
- Иконки — SVG (lucide/boxicons), один стиль и толщина; **не emoji** как иконки.
- ⚠️ Сейчас карточки товаров/курсов показывают emoji-заглушки (нет S3-картинок локально) — заменить реальными изображениями (данные/ассеты, не CSS).
