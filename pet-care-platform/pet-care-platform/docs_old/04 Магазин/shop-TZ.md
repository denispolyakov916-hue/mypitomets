# Техническое задание: Магазин

## 1. Общие сведения

### 1.1. Наименование проекта
Интернет-магазин товаров для животных платформы "Питомец+"

### 1.2. Цель проекта
Создать полнофункциональный магазин с персонализированными рекомендациями

### 1.3. Задачи проекта
- Разработать каталог товаров
- Реализовать поиск и фильтрацию
- Интегрировать персонализированные рекомендации на основе PetID
- Создать систему корзины и оформления заказов
