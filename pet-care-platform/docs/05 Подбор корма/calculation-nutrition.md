## Описание

### Назначение функции
Универсальная функция `calculate_daily_calories()` предназначена для расчета стабильной дневной калорийности рациона питания домашних животных (кошек и собак) на основе индивидуальных параметров питомца, его физиологического состояния, уровня активности и внешних факторов.

Функция обеспечивает:
- **Персонализированный расчет** калорийности на основе веса, возраста, породы и здоровья животного
- **Динамическое планирование** на 14 дней с учетом ежедневных активностей
- **Адаптацию к изменениям** в расписании и состоянии питомца
- **Валидацию данных** на соответствие ветеринарным нормам

### Сертификация и источники данных
Все используемые коэффициенты и формулы основаны на рекомендациях авторитетных ветеринарных организаций:

**Основные источники:**
- **National Research Council (NRC)**: "Nutrient Requirements of Dogs and Cats" (2006, обновлено 2021)
  - Ссылка: https://nap.nationalacademies.org/catalog/10668/nutrient-requirements-of-dogs-and-cats
- **World Small Animal Veterinary Association (WSAVA)**: Global Nutrition Guidelines (2021)
  - Ссылка: https://wsava.org/wp-content/uploads/2021/01/WSAVA-Global-Nutrition-Guidelines-2021.pdf
- **Association of American Feed Control Officials (AAFCO)**: Official Publication (2023)
  - Ссылка: https://www.aafco.org/publications/official-publication/
- **European Society of Veterinary and Comparative Nutrition (ESVCN)**: Guidelines (2022)
  - Ссылка: https://esvcn.eu/wp-content/uploads/2022/06/ESVCN-nutrition-guidelines-2022.pdf

**Дополнительные исследования:**
- Fascetti AJ, Delaney SJ. (2012). Applied Veterinary Clinical Nutrition
- Laflamme DP. (2014). Understanding pet obesity and its many causes
- Butterwick RF, Hawthorne AJ. (1998). Advances in dietary management of obesity in dogs and cats

### Актуальность данных
- **Основные формулы (RER/DER)**: Подтверждены исследованиями 2006-2021 гг.
- **Коэффициенты активности**: Обновлены на основе мета-анализов 2015-2021 гг.
- **Возрастные коэффициенты**: Уточнены по данным долгосрочных исследований (2010-2023)
- **Коэффициенты здоровья**: Основаны на клинических исследованиях заболеваний (2018-2023)

## Логика работы функции

### Основной алгоритм
```
RER = 70 × (вес_в_кг)^0.75  # Resting Energy Requirement
DER = RER × K₁ × K₂ × K₃ × ... × Kₙ  # Daily Energy Requirement
```

где K₁, K₂, K₃ - последовательные коэффициенты различных факторов.

### Двухнедельное планирование
Функция рассчитывает индивидуальный рацион на 14 дней с учетом:

**Ежедневные активности:**
- Время прогулок (минуты/день)
- Интенсивность активности (бег, игры, тренировки)
- Частота активностей (раз/день)
- Тип активности (прогулка, бег, плавание, охота)

**Внешние факторы:**
- Климатические условия (температура, влажность)
- Тип шерсти (длина, густота)
- Условия содержания (квартира, дом, улица)
- Стресс-факторы (новые ситуации, шум, одиночество)

### Адаптивная логика
- **Базовый расчет**: Определяется по статическим параметрам (возраст, порода, здоровье)
- **Динамическая коррекция**: Применяется ежедневно на основе запланированных активностей
- **Валидация**: Проверка на соответствие безопасным диапазонам (±20% от базового)
- **Рекомендации**: Предложения по коррекции при выходе за допустимые пределы

### Порядок применения коэффициентов
1. **Базовые коэффициенты** (возраст, размер породы)
2. **Здоровье** (заболевания, физиологическое состояние)
3. **Активность** (ежедневная + запланированная)
4. **Внешние факторы** (климат, шерсть, условия)
5. **Коррекция** (стресс, реабилитация)

## Примеры расчета

### Пример 1: Взрослый лабрадор-ретривер
**Параметры:**
- Вес: 32 кг
- Возраст: 4 года
- Активность: средняя (1.5 часа прогулок ежедневно)
- Здоровье: здоров
- Климат: умеренный
- Шерсть: короткая

**Расчет:**
```
RER = 70 × (32)^0.75 = 70 × 9.89 ≈ 692 ккал
DER = 692 × 1.0 (средний размер) × 1.4 (взрослый) × 1.4 (активность) = 1355 ккал/день
```

### Пример 2: Пожилая британская кошка с ожирением
**Параметры:**
- Вес: 5.5 кг
- Возраст: 12 лет
- Статус: стерилизована
- Здоровье: ожирение (стадия 2)
- Активность: домашняя

**Расчет:**
```
RER = 70 × (5.5)^0.75 = 70 × 2.89 ≈ 202 ккал
DER = 202 × 1.0 (средний размер) × 0.9 (пожилая) × 1.2 (стерилизована) × 0.7 (ожирение) = 153 ккал/день
```

### Пример 3: Щенок золотистого ретривера с запланированными активностями
**Параметры:**
- Вес: 15 кг
- Возраст: 4 месяца
- Активность: базовая + 30 мин бега ежедневно
- Здоровье: здоров

**Расчет:**
```
Базовый RER = 70 × (15)^0.75 = 70 × 5.03 ≈ 352 ккал
Базовый DER = 352 × 2.5 (возраст) = 880 ккал

Динамическая коррекция за активность:
- 30 мин бега = +15% к базовому = 132 ккал
- Итоговый дневной рацион = 1012 ккал
```

## Техническое задание для реализации

### Требования к функции
```typescript
interface PetData {
  species: 'dog' | 'cat';
  weight: number; // кг
  age: number; // месяцы
  breed_size?: 'toy' | 'small' | 'medium' | 'large' | 'giant';
  health_conditions: HealthCondition[];
  activity_level: ActivityLevel;
  sterilization_status?: boolean;
  coat_type?: CoatType;
  living_conditions?: LivingConditions;
  planned_activities?: DailyActivity[];
}

interface DailyActivity {
  type: 'walk' | 'run' | 'play' | 'training' | 'swimming';
  duration_minutes: number;
  intensity: 'low' | 'medium' | 'high';
  frequency_per_day: number;
}

function calculate_daily_calories(petData: PetData): {
  base_calories: number;
  adjusted_calories: number;
  weekly_plan: DailyCalories[];
  warnings: string[];
  recommendations: string[];
}
```

### Валидация и безопасность
- **Диапазоны веса**: 0.5-80 кг для собак, 0.5-10 кг для кошек
- **Безопасные пределы**: ±30% от расчетного значения
- **Предупреждения**: При выходе за допустимые диапазоны
- **Рекомендации**: Консультация ветеринара при сложных случаях

### Тестирование
- Unit-тесты для каждой категории коэффициентов
- Интеграционные тесты с реальными кейсами
- Валидация на соответствие NRC/WSAVA стандартам

### Производительность
- Кеширование рассчитанных значений
- Предварительные вычисления для популярных комбинаций
- Оптимизация для мобильных устройств
