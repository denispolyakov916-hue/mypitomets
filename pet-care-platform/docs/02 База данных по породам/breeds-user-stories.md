# User Stories - База данных по породам

## Обзор
База данных по породам предоставляет персонализированные рекомендации для владельцев собак и кошек, интегрируясь со всеми модулями платформы "Питомец+".

---

## 🎯 EPIC: Создание профиля PetID с породой

### US-1.1: Выбор породы при создании профиля питомца
**Как** новый пользователь, создающий профиль питомца,  
**Я хочу** выбрать породу из списка доступных,  
**Чтобы** получить персонализированные рекомендации с самого начала.

#### Критерии приёмки:
- Список пород отсортирован по популярности (собаки и кошки отдельно)
- Доступен поиск по названию породы
- Показывается краткое описание каждой породы
- После выбора породы автоматически заполняются базовые параметры

#### Сценарии тестирования:
**Сценарий: Успешный выбор породы**
- Дано: Пользователь на странице создания PetID
- Когда: Выбирает "Лабрадор" из списка
- Тогда: Автоматически заполняются вес 25-36 кг, уровень энергии "высокий"

**Сценарий: Поиск породы**
- Дано: Открыт список пород
- Когда: Пользователь вводит "лабрадор"
- Тогда: Показывается "Лабрадор Ретривер" в результатах поиска

---

### US-1.2: Автозаполнение параметров на основе породы
**Как** владелец питомца, указавший породу,  
**Я хочу** автоматически получить рекомендуемые параметры,  
**Чтобы** не заполнять всё вручную и избежать ошибок.

#### Критерии приёмки:
- Вес: диапазон нормы для породы
- Активность: рекомендуемый уровень
- Характер: типичные черты поведения
- Здоровье: известные риски породы
- Уход: частота груминга

#### Сценарии тестирования:
**Сценарий: Автозаполнение для лабрадора**
- Дано: Выбрана порода "Лабрадор"
- Когда: Система загружает данные породы
- Тогда: Предзаполняются: вес 25-36 кг, активность "высокая", груминг "еженедельный"

---

## 🔍 EPIC: Просмотр информации о породах

### US-2.1: Каталог пород с фильтрами
**Как** пользователь, интересующийся породами,  
**Я хочу** просмотреть каталог пород с возможностью фильтрации,  
**Чтобы** найти подходящую породу по своим критериям.

#### Критерии приёмки:
- Фильтры: вид (собака/кошка), размер, гипоаллергенность
- Фильтры: подходит для квартиры, для новичков
- Сортировка: по популярности, алфавиту
- Поиск по названию породы

#### Сценарии тестирования:
**Сценарий: Фильтр по размеру**
- Дано: Открыт каталог пород
- Когда: Выбираю фильтр "маленький размер" и "подходит для квартиры"
- Тогда: Показываются только маленькие породы, подходящие для квартиры

### US-2.2: Детальная информация о породе
**Как** пользователь, выбравший породу,  
**Я хочу** прочитать подробную информацию о характеристиках,  
**Чтобы** понять особенности ухода и содержания.

#### Критерии приёмки:
- Физические характеристики (вес, рост, продолжительность жизни)
- Характер и поведение (активность, обучаемость, социальность)
- Здоровье (генетические риски, профилактика)
- Уход (груминг, питание, процедуры)
- Совместимость (с детьми, другими животными)

#### Сценарии тестирования:
**Сценарий: Просмотр рисков здоровья**
- Дано: Открыта страница породы "Золотистый ретривер"
- Когда: Перехожу к разделу "Здоровье"
- Тогда: Вижу список генетических заболеваний с процентами распространенности

## 📊 EPIC: Сравнение питомца с эталоном породы

### US-3.1: Полное сравнение параметров питомца
**Как** владелец питомца с известной породой,  
**Я хочу** сравнить параметры своего питомца с эталоном породы,  
**Чтобы** понять, насколько мой питомец соответствует стандартам.

#### Критерии приёмки:
- Анализ веса: норма/избыток/недостаток
- Оценка активности: соответствует/недостаточно/избыточно
- Проверка поведения: типичные черты
- Оценка здоровья: текущие проблемы + риски породы
- Совместимость с условиями содержания

#### Сценарии тестирования:
**Сценарий: Идеально здоровый питомец**
- Дано: Пит весит 30 кг, активность средняя, нет проблем здоровья
- Когда: Сравниваю с эталоном лабрадора (25-36 кг, активность высокая)
- Тогда: Получаю общий скор 85/100 с рекомендациями увеличить активность

### US-3.2: Персонализированные рекомендации
**Как** владелец питомца, получивший анализ,  
**Я хочу** увидеть конкретные рекомендации по улучшению,  
**Чтобы** знать, что делать для здоровья питомца.

#### Критерии приёмки:
- Рекомендации по весу (диета, активность)
- Советы по питанию (тип корма, частота кормления)
- Рекомендации по здоровью (профилактика, осмотры)
- Советы по уходу (груминг, процедуры)
- Предложения по дрессировке

#### Сценарии тестирования:
**Сценарий: Рекомендации для перекормленного питомца**
- Дано: Пит весит 40 кг (эталон лабрадора 25-36 кг)
- Когда: Получаю анализ
- Тогда: Рекомендации включают "Уменьшить порции на 10-15%", "Диетический корм"

## 🛒 EPIC: Персонализация покупок

### US-4.1: Рекомендации корма по породе
**Как** пользователь, выбирающий корм в магазине,  
**Я хочу** видеть рекомендации, основанные на породе питомца,  
**Чтобы** выбрать оптимальный корм для здоровья.

#### Критерии приёмки:
- Показываются корма, подходящие для размера породы
- Учитываются потребности в белке и калориях
- Исключаются аллергены породы
- Учитывается тип питания (сухой/влажный/смешанный)

#### Сценарии тестирования:
**Сценарий: Подбор корма для щенка лабрадора**
- Дано: Пользователь ищет корм для щенка лабрадора
- Когда: Система анализирует потребности породы
- Тогда: Рекомендуются корма с высоким содержанием белка и калорий

### US-4.2: Подбор товаров по характеристикам породы
**Как** пользователь в магазине,  
**Я хочу** видеть персонализированные предложения товаров,  
**Чтобы** купить то, что действительно нужно моей породе.

#### Критерии приёмки:
- Игрушки по уровню активности
- Средства ухода по типу шерсти
- Аксессуары по размеру
- Профилактика по рискам здоровья

#### Сценарии тестирования:
**Сценарий: Рекомендации для длинношерстной породы**
- Дано: Порода "Колли" (длинная шерсть, регулярный груминг)
- Когда: Пользователь в разделе "Уход за шерстью"
- Тогда: Показываются расчески, шампуни для длинношерстных, щетки

## 📚 EPIC: Персонализация обучения

### US-5.1: Рекомендации курсов по обучаемости
**Как** пользователь, выбирающий курсы дрессировки,  
**Я хочу** видеть рекомендации, учитывающие обучаемость породы,  
**Чтобы** выбрать подходящий уровень сложности.

#### Критерии приёмки:
- Курсы по уровню обучаемости (высокий/средний/низкий)
- Учитывается опыт дрессировки питомца
- Рекомендуется продолжительность обучения
- Предлагаются курсы по специфическим проблемам породы

#### Сценарии тестирования:
**Сценарий: Курсы для лабрадора с базовым опытом**
- Дано: Лабрадор с высокой обучаемостью, базовый опыт дрессировки
- Когда: Пользователь смотрит курсы
- Тогда: Рекомендуются продвинутые курсы послушания

## 📅 EPIC: Календарь ухода по породе

### US-6.1: Автоматическая генерация напоминаний
**Как** владелец питомца с известной породой,  
**Я хочу** автоматически получать напоминания об уходе,  
**Чтобы** не забывать важные процедуры.

#### Критерии приёмки:
- Напоминания о вакцинации по графику
- Уход за шерстью по частоте груминга
- Профилактические осмотры по рискам породы
- Чистка зубов по рекомендациям породы

#### Сценарии тестирования:
**Сценарий: Напоминания для брахицефальной породы**
- Дано: Порода "Мопс" (брахицефал, нуждается в охлаждении)
- Когда: Наступает лето
- Тогда: Появляются напоминания "Избегать жары", "Следить за дыханием"

### US-6.2: Сезонные рекомендации по уходу
**Как** владелец, живущий в регионе с сезонами,  
**Я хочу** получать сезонные советы по уходу,  
**Чтобы** адаптировать заботу к погоде.

#### Критерии приёмки:
- Летние советы для брахицефалов
- Зимний уход за шерстью
- Весенние процедуры против паразитов
- Осенние рекомендации по питанию

## ⚕️ EPIC: Профилактика здоровья

### US-7.1: Предупреждения о рисках породы
**Как** владелец, заботящийся о здоровье питомца,  
**Я хочу** знать о генетических рисках породы,  
**Чтобы** вовремя принимать меры профилактики.

#### Критерии приёмки:
- Список заболеваний с процентами распространенности
- Возраст проявления рисков
- Рекомендации по профилактике
- График скрининговых обследований

#### Сценарии тестирования:
**Сценарий: Риски дисплазии для крупной породы**
- Дано: Порода "Немецкая овчарка" (риск дисплазии тазобедренных суставов 25%)
- Когда: Пит достигает 12 месяцев
- Тогда: Появляется рекомендация "Рентген на дисплазию"

### US-7.2: Персонализированные медицинские рекомендации
**Как** владелец, планирующий визит к ветеринару,  
**Я хочу** получить чек-лист обследований для породы,  
**Чтобы** обсудить со специалистом все важные аспекты.

#### Критерии приёмки:
- Базовые обследования для возраста
- Специфические тесты по рискам породы
- Частота осмотров
- Профилактические меры

## Приоритезация по MoSCoW

### Must Have (Обязательно):
- US-1.1, US-1.2: Базовый выбор и автозаполнение породы
- US-2.1, US-2.2: Просмотр информации о породах
- US-3.1, US-3.2: Сравнение питомца с эталоном
- US-4.1: Рекомендации корма по породе

### Should Have (Желательно):
- US-6.1: Напоминания по уходу
- US-7.1: Предупреждения о рисках здоровья
- US-5.1: Рекомендации курсов

### Could Have (Возможно):
- US-8.1: Мобильная оптимизация
- US-10.1: Аналитика популярности
- US-4.2: Расширенные рекомендации товаров

### Won't Have (Не в этом релизе):
- Интеграция с ветеринарными клиниками
- Социальные функции (отзывы о породах)
- Сравнение нескольких пород
