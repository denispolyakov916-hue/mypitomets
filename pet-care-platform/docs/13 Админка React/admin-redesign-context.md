# Контекст задачи: Редизайн админ-панели Питомец+

## Обзор проекта

**Питомец+** — комплексная экосистема для владельцев домашних животных, построенная вокруг единого цифрового профиля питомца (PetID). Проект включает 8 основных приложений Django с богатой админ-панелью.

## Текущая структура админ-панели

### Существующие кастомизации

#### 1. Кастомные шаблоны (templates/admin/)
- `dashboard.html` - Главный дашборд с метриками
- `pet_analytics.html` - Аналитика PetID
- `payment_analytics.html` - Аналитика платежей
- `recommendation_settings.html` - Настройки рекомендаций

#### 2. Кастомные AdminSite
- `PetCareAdminSite` в `shop/admin_site.py` - расширенное меню с аналитикой
- Кастомные заголовки и навигация

#### 3. Admin Views (`shop/admin_views.py`)
- `admin_dashboard()` - дашборд с 20+ метриками
- `payment_analytics()` - аналитика платежей
- `pet_analytics()` - аналитика PetID
- `recommendation_settings()` - настройки рекомендаций

### Структура моделей (8 приложений)

#### Users (users/models.py)
- `User` - основной пользователь с UUIDv7
- `Token` - JWT токены для аутентификации

#### Pets (pets/models.py)
- `Pet` - основной профиль питомца с PetID
- `Reminder` - система напоминаний по уходу

#### Shop (shop/models.py)
- `Product` - товары с расширенной фильтрацией
- `Cart/CartItem` - корзина покупок
- `Order/OrderItem` - заказы
- `Reservation` - бронирование товаров
- `Address` - адреса доставки
- `Return` - система возвратов

#### Training (training/models.py)
- `Course` - образовательные курсы
- `Lesson` - уроки (унаследованная архитектура)
- `UserCourse` - приобретенные курсы
- `UserCourseProgress` - прогресс обучения
- `Comment/CommentLike` - система комментариев
- `Rating` - оценки курсов

#### Payments (payments/models.py)
- `Payment` - платежи с поддержкой статусов

#### Reviews (reviews/models.py)
- `Review` - отзывы для товаров и курсов

#### Calendar (calendar/models.py)
- `CalendarEvent` - события календаря
