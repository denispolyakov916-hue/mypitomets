## 1. Основные таблицы базы знаний (связанные с породами)

### 1.1 Таблица "Породы собак и кошек" (breeds) {#11-таблица-породы-собак-и-кожек-breeds}

**Назначение:** Основная таблица базы знаний, содержащая информацию о всех породах собак и кошек, доступных в системе.

**Хранимые данные:**
- Уникальный идентификатор породы (integer)
- Тип животного (собака/кошка)
- Название породы (На русском)
- Оригинальное название породы (На английском)
- Подробное описание породы
- Краткое описание породы
- Размер (миниатюрный/маленький/средний/большой/гигантский)
- Средний вес (мин/макс)
- Средняя продолжительность жизни
- Уровень активности
- Уровень ухода (низкий/средний/высокий)
- Особенности шерсти (длина, тип, частота линьки)
- Темперамент и поведение
- Страна происхождения


**Связи:**
- Многие ко многим с таблицей аллергий (allergies) - для определения типичных аллергий породы
- Многие ко многим с таблицей хронических заболеваний (chronic_diseases) - для определения предрасположенностей
- Многие ко многим с таблицей проблем здоровья (health_issues) - для определения типичных проблем
- Многие ко многим с таблицей вакцинаций (vaccinations) - для определения стандартного графика вакцинаций
- Многие ко многим с таблицей препаратов (medications) - для определения рекомендуемых препаратов
- Один ко многим с таблицей PetID - для привязки питомцев к породам


### 1.2 Таблица "Аллергии" (allergies) {#12-таблица-аллергии-allergies}

**Назначение:** База знаний по типичным аллергическим реакциям у собак и кошек различных пород.

**Хранимые данные:**
- Уникальный идентификатор аллергии
- Название аллергии
- Тип аллергии (пищевая/контактная/ингаляционная/лекарственная/паразитарная)
- Симптомы аллергии
- Возможные аллергены
- Степень распространенности
- Потенциальная опасность (низкая/средняя/высокая/критическая)
- Возрастные особенности проявления
- Сезонность проявления
- Методы диагностики
- Рекомендации по лечению и профилактике

**Связи:**
- Многие ко многим с таблицей пород (breeds) - аллергия может быть характерна для нескольких пород
- Многие ко многим с таблицей PetID - у конкретного питомца может быть диагностирована аллергия

### 1.3 Таблица "Хронические заболевания" (chronic_diseases) {#13-таблица-хронические-заболевания-chronic_diseases}

**Назначение:** База знаний по хроническим заболеваниям, к которым предрасположены различные породы собак и кошек.

**Хранимые данные:**
- Уникальный идентификатор заболевания
- Название заболевания
- Тип животного (собака/кошка/оба)
- Возраст проявления (молодой/взрослый/пожилой)
- Генетическая предрасположенность
- Симптомы и признаки
- Методы диагностики
- Возможные осложнения
- Рекомендации по лечению
- Профилактические меры
- Прогноз и качество жизни
- Частота проявления у породы

**Связи:**
- Многие ко многим с таблицей пород (breeds) - заболевание может быть характерно для нескольких пород
- Многие ко многим с таблицей PetID - у конкретного питомца может быть диагностировано заболевание
- Многие ко многим с таблицей препаратов (medications) - рекомендуемые препараты для лечения


### 1.4 Таблица "Проблемы здоровья" (health_issues) {#14-таблица-проблемы-здоровья-health_issues}

**Назначение:** База знаний по типичным проблемам здоровья, с которыми сталкиваются собаки и кошки различных пород.

**Хранимые данные:**
- Уникальный идентификатор проблемы
- Название проблемы
- Категория (глаза/уши/кости/суставы/кожа/пищеварение/мочеполовая система/дыхание/поведение)
- Возможные причины
- Симптомы и признаки
- Возрастные особенности
- Сезонные факторы
- Методы диагностики
- Способы лечения
- Профилактика
- Срочность обращения к ветеринару

**Связи:**
- Многие ко многим с таблицей пород (breeds) - проблема может быть характерна для нескольких пород
- Многие ко многим с таблицей PetID - история проблем конкретного питомца
- Многие ко многим с таблицей хронических заболеваний - проблемы могут переходить в хронические состояния

### 1.5 Таблица "Вакцинации" (vaccinations) {#15-таблица-вакцинации-vaccinations}

**Назначение:** База знаний по обязательным и рекомендуемым вакцинациям для различных пород собак и кошек.

**Хранимые данные:**
- Уникальный идентификатор вакцинации
- Название вакцины
- Тип вакцины (живые/инактивированные/рекомбинантные)
- Против каких заболеваний защищает
- Возраст первой вакцинации
- Схема вакцинаций (первичная/ревакцинация)
- Интервалы между вакцинациями
- Обязательность (обязательная/рекомендуемая)
- Возможные побочные эффекты
- Противопоказания
- Особенности для разных пород

**Связи:**
- Многие ко многим с таблицей пород (breeds) - рекомендуемые вакцины для каждой породы
- Многие ко многим с таблицей PetID - история вакцинаций конкретного питомца
- Многие ко многим с таблицей хронических заболеваний - вакцины могут быть противопоказаны при некоторых заболеваниях

### 1.6 Таблица "Принимаемые препараты" (medications) {#16-таблица-принимаемые-препараты-medications}

**Назначение:** База знаний по лекарственным препаратам, которые могут назначаться собакам и кошкам различных пород.

**Хранимые данные:**
- Уникальный идентификатор препарата
- Название препарата (торговое и действующее вещество)
- Форма выпуска (таблетки/капли/инъекции/мази/суспензии)
- Показания к применению
- Противопоказания
- Возможные побочные эффекты
- Дозировка по весу/возрасту
- Курс лечения
- Особенности применения для разных пород
- Взаимодействие с другими препаратами
- Условия хранения

**Связи:**
- Многие ко многим с таблицей пород (breeds) - рекомендуемые препараты для каждой породы
- Многие ко многим с таблицей PetID - препараты, назначенные конкретному питомцу
- Многие ко многим с таблицей хронических заболеваний - препараты для лечения конкретных заболеваний
- Многие ко многим с таблицей аллергий - препараты могут вызывать аллергические реакции

**Функциональное назначение полей:**
- Используется для безопасного назначения препаратов с учетом особенностей породы
- Помогает избегать нежелательных взаимодействий лекарств
- Данные используются в системе управления лечением питомца

### 1.7 Таблица "Стандартные процедуры ухода" (care_procedures) {#17-таблица-стандартные-процедуры-ухода-care_procedures}

**Назначение:** База знаний по процедурам ухода, которые являются типичными и рекомендуемыми для различных пород собак и кошек.

**Хранимые данные:**
- Уникальный идентификатор процедуры
- Название процедуры
- Категория процедуры (гигиена/уход за шерстью/уход за когтями/уход за зубами/уход за ушами)
- Тип животного (собака/кошка/оба)
- Частота выполнения (ежедневно/еженедельно/ежемесячно/по необходимости)
- Возрастные ограничения
- Сезонные особенности
- Необходимые инструменты и средства
- Время выполнения процедуры
- Уровень сложности (простой/средний/сложный)
- Рекомендации по подготовке питомца
- Возможные риски и противопоказания
- Признаки правильного выполнения

**Связи:**
- Многие ко многим с таблицей пород (breeds) - процедуры, рекомендуемые для каждой породы
- Многие ко многим с таблицей PetID - история выполненных процедур конкретного питомца

**Функциональное назначение полей:**
- Используется для создания персонального календаря ухода за питомцем
- Помогает владельцам правильно ухаживать за питомцами разных пород
- Данные используются в системе напоминаний о процедурах ухода

### 1.8 Таблица "Поведенческие особенности" (behavioral_traits) {#18-таблица-поведенческие-особенности-behavioral_traits}

**Назначение:** База знаний по характерным чертам поведения, темпераменту и особенностям дрессировки различных пород собак и кошек.

**Хранимые данные:**
- Уникальный идентификатор особенности
- Название особенности поведения
- Тип поведения (положительное/нейтральное/проблемное)
- Уровень выраженности (низкий/средний/высокий)
- Возраст проявления
- Влияющие факторы (окружающая среда/воспитание/генетика)
- Методы коррекции поведения
- Рекомендации по дрессировке
- Совместимость с другими животными
- Совместимость с детьми
- Особенности социализации
- Признаки проблемного поведения

**Связи:**
- Многие ко многим с таблицей пород (breeds) - поведенческие черты, характерные для каждой породы
- Один ко многим с таблицей PetID - привязка к конкретному питомцу для отслеживания индивидуальных особенностей

**Функциональное назначение полей:**
- Используется для подбора подходящей породы под условия содержания
- Помогает в воспитании и дрессировке питомцев
- Данные используются для рекомендаций по поведенческой коррекции

### 1.9 Таблица "Профилактические мероприятия" (preventive_measures) {#19-таблица-профилактические-мероприятия-preventive_measures}

**Назначение:** База знаний по рекомендациям профилактики заболеваний и проблем здоровья для различных пород собак и кошек.

**Хранимые данные:**
- Уникальный идентификатор мероприятия
- Название мероприятия
- Тип профилактики (вакцинация/дегельминтизация/обработка от паразитов/диспансеризация)
- Возрастные группы (щенки/молодые/взрослые/пожилые)
- Сезонность проведения
- Частота выполнения
- Необходимые препараты и инструменты
- Стоимость мероприятия
- Продолжительность эффекта
- Возможные побочные эффекты
- Показания и противопоказания
- Рекомендации по подготовке


**Связи:**
- Многие ко многим с таблицей пород (breeds) - профилактические мероприятия для каждой породы
- Многие ко многим с таблицей хронических заболеваний (chronic_diseases) - профилактика конкретных заболеваний
- Многие ко многим с таблицей PetID - история проведенных профилактических мероприятий

**Функциональное назначение полей:**
- Используется для создания персонального плана профилактики заболеваний
- Помогает своевременно проводить необходимые мероприятия
- Данные используются в ветеринарном календаре и системе напоминаний

### 1.10 Таблица "Возрастные особенности" (age_characteristics) {#110-таблица-возрастные-особенности-age_characteristics}

**Назначение:** База знаний по особенностям развития, изменениям в здоровье и поведении собак и кошек разных пород на различных этапах жизни.

**Хранимые данные:**
- Уникальный идентификатор характеристики
- Возрастной период (щенок/молодой/взрослый/пожилой/старик)
- Диапазон возраста (в месяцах/годах)
- Физические изменения
- Поведенческие изменения
- Изменения в здоровье
- Особенности питания
- Уровень активности
- Потребность в уходе
- Риск развития заболеваний
- Рекомендации по уходу
- Признаки нормального развития
- Признаки отклонений

**Связи:**
- Многие ко многим с таблицей пород (breeds) - возрастные особенности каждой породы
- Один ко многим с таблицей PetID - привязка к возрасту конкретного питомца

**Функциональное назначение полей:**
- Используется для мониторинга правильного развития питомца
- Помогает адаптировать уход под возрастные особенности
- Данные используются для персонализированных рекомендаций по уходу

### 1.11 Таблица "История проблем здоровья" (pet_health_history) {#111-таблица-история-проблем-здоровья-pet_health_history}

**Назначение:** История всех заболеваний, проблем со здоровьем и медицинских вмешательств конкретного питомца.

**Хранимые данные:**
- Уникальный идентификатор записи
- Дата и время возникновения проблемы
- Тип проблемы (заболевание/травма/профилактика)
- Подробное описание симптомов
- Диагноз (если установлен)
- Причина возникновения
- Методы диагностики
- Назначенное лечение
- Результат лечения
- Продолжительность лечения
- Рекомендации врача
- Стоимость лечения
- Признаки выздоровления
- Последствия для здоровья

**Связи:**
- Многие ко многим с таблицей PetID - привязка к конкретному питомцу
- Многие ко одному с таблицей проблем здоровья (health_issues) - ссылка на тип проблемы из базы знаний
- Многие ко многим с таблицей препаратов (medications) - назначенные препараты
- Многие ко многим с таблицей вакцинаций (vaccinations) - проведенные вакцинации

**Функциональное назначение полей:**
- Используется для ведения полной медицинской истории питомца
- Помогает отслеживать эффективность лечения и динамику здоровья
- Данные используются для прогнозирования рисков и персонализированных рекомендаций
