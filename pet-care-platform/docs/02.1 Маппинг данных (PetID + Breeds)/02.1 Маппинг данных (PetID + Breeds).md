## 1. Введение и назначение

### 1.1. Назначение документа

Этот документ описывает полную спецификацию маппинга данных между системой PetID (профили питомцев) и базой данных пород (Breeds), а также механизмы персонализации товаров и курсов на основе этих данных.

### 1.2. Основные компоненты

- **PetID**: Центральная сущность платформы, содержащая всю информацию о питомце
- **Breeds**: Справочник характеристик пород собак и кошек (128 пород)
- **PersonalizationService**: Сервис персонализированных рекомендаций
- **Shop**: Магазин товаров с персонализацией
- **Training**: Система курсов с персонализацией

### 1.3. Ключевые связи

```
User (владелец) → Pet (питомец) ←→ Breed (порода)
                                      ↓
PersonalizationService → Shop (товары) | Training (курсы)
```

## 2. Архитектура и компоненты

### 2.1. Модели данных

#### Pet (backend/apps/pets/models.py)
Основная модель профиля питомца с расширенными полями персонализации.

#### Breed (backend/apps/pets/breed_models.py)
Справочник пород с характеристиками, рисками здоровья, питанием и уходом.

#### Связанные таблицы пород:
- **BreedHealth**: Генетические риски здоровья
- **BreedNutrition**: Рекомендации по питанию
- **BreedCare**: Процедуры ухода

### 2.2. Сервисы персонализации

#### PersonalizationService (backend/apps/pets/services.py)
Центральный сервис для формирования персонализированных рекомендаций.

#### Основные методы:
- `get_context(user)`: Получение контекста персонализации
- `get_pet_context(user, pet_id)`: Контекст конкретного питомца
- `get_product_recommendations(user)`: Рекомендации товаров
- `get_course_recommendations(user)`: Рекомендации курсов

### 4.3. Алгоритмы персонализации товаров

#### Основные правила фильтрации:
1. **Фильтр по виду**: Только товары для собак или кошек
2. **Исключение аллергии**: Полное исключение аллергенов
3. **Возрастная адаптация**: Puppy/Kitten → junior товары, Senior → mature
4. **Здоровье**: overweight → diet/light, skin_issues → special shampoos
5. **Активность**: high energy → performance foods, low → calm toys

### 5.3. Алгоритмы персонализации курсов

#### Основные правила фильтрации:
1. **Фильтр по виду**: Только курсы для собак, кошек или универсальные

4. **Возраст**: Молодые → базовые курсы, пожилые → адаптированные программы

```python
@property
def age(self):
    """Возраст питомца в годах."""
    if not self.date_of_birth:
        return None

    from datetime import date
    today = date.today()
    age = today.year - self.date_of_birth.year

    # Проверка прошедшего дня рождения
    if today.month < self.date_of_birth.month or \
       (today.month == self.date_of_birth.month and today.day < self.date_of_birth.day):
        age -= 1

    return max(0, age)
```

```python
def calculate_profile_completeness(self):
    """
    Расчет процента заполненности профиля.
    """
    # Обязательные поля (50% веса)
    required_fields = [
        self.name, self.species, self.date_of_birth,
        self.breed, self.weight, self.gender != 'unknown',
    ]

    # Поля здоровья (20% веса)
    health_fields = [
        bool(self.health_issues),
        bool(self.allergies) or bool(self.excluded_ingredients),
        self.activity_level,
    ]

    # Поля поведения (10% веса)
    behavior_fields = [
        bool(self.behavioral_problems) or self.behavior_type,
    ] if self.species in ['dog', 'cat'] else []

    # Опциональные поля (20% веса)
    optional_fields = [
        self.is_neutered is not None,
        bool(self.photo),
        self.behavior_type,
        self.social_level,
        self.training_experience,
        self.diet_type,
        self.housing_type if self.species == 'dog' else self.activity_level,
        self.has_yard,
        self.has_children,
        self.other_pets,
        self.walk_frequency if self.species == 'dog' else None,
        self.walk_duration if self.species == 'dog' else None,
        bool(self.owner_phone) or bool(self.owner_email),
    ]

    # Расчет
    required_filled = sum(1 for f in required_fields if f)
    health_filled = sum(1 for f in health_fields if f)
    behavior_filled = sum(1 for f in behavior_fields if f) if behavior_fields else 1
    optional_filled = sum(1 for f in optional_fields if f is not None)

    # Веса категорий
    required_score = (required_filled / max(len(required_fields), 1)) * 50
    health_score = (health_filled / max(len(health_fields), 1)) * 20
    behavior_score = (behavior_filled / max(len(behavior_fields), 1)) * 10 if behavior_fields else 10
    optional_score = (optional_filled / max(len([f for f in optional_fields if f is not None]), 1)) * 20

    total = required_score + health_score + behavior_score + optional_score
    return min(100, int(total))
```
