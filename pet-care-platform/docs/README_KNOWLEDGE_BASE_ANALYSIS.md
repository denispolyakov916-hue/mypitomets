### 2. **FOOD_RECOMMENDATION_ALGORITHM.md**
**Детальный алгоритм подбора корма**

**Содержание:**
- Расчет калорий (RER, DER)
- Расчет БЖУ (белок, жир, углеводы)
- Алгоритм подбора корма (10 факторов)
- Расчет порций и частоты кормления
- Примеры для разных сценариев:
  - Щенок Лабрадора (4 месяца, высокая активность)
  - Кошка с аллергией (5 лет, стерилизованная)
  - Пожилая собака с избыточным весом (10 лет, дисплазия)

**Формулы:**
```python
RER = 70 * (вес в кг)^0.75
DER = RER * возраст * активность * стерилизация * вес_корректировка * порода

Белок (грамм) = (DER * protein_percent / 100) / 4
Жир (грамм) = (DER * fat_percent / 100) / 9
```

**Читать, если:** Нужно реализовать алгоритм подбора корма и расчета рациона.

## 🎯 Ключевые выводы

### Что нужно для персонализации:

#### 1. **Магазин** (подбор корма и товаров)
**Из базы знаний:**
- breeds: size_category, weight_min/max, energy_level, health_risk_level, grooming_frequency
- breed_health: condition_name, severity, affected_system
- breed_nutrition: protein_need, diet_type, allergen_name
- breed_care: care_category, tools_needed

**Из PetID:**
- species, breed, weight, date_of_birth, activity_level, health_issues, allergies

**Из товаров (КРИТИЧНО):**
- nutrition_params: БЖУ, калории, ингредиенты, аллергены, health_benefits

#### 2. **Подбор корма** (рацион и план питания)
**Алгоритм:**
1. Расчет базовых калорий: `RER = 70 * (вес^0.75)`
2. Корректировка по 6 факторам: возраст, активность, стерилизация, вес, порода, здоровье
3. Расчет БЖУ: белок (25-35%), жир (12-18%), углеводы (остаток)
4. Фильтрация кормов: вид → возраст → размер → аллергены → здоровье → порода
5. Скоринг релевантности (0-100 баллов)
6. Расчет порций: дневная → по приемам пищи

#### 3. **PetID** (сравнение с эталонами)
**Сравниваемые параметры:**
- Вес (текущий vs идеальный) → рекомендации по диете
- Активность (фактическая vs требуемая) → рекомендации по прогулкам
- Здоровье (текущее vs риски породы) → предупреждения и скрининги
- Поведение (проблемы vs типичное) → рекомендации по дрессировке

#### 4. **Дневник здоровья** (автогенерация событий)
**Генерация событий:**
- Вакцинации (из breed_medical + common_vaccinations)
- Уход (из breed_care: груминг, чистка зубов, когтей)
- Ветосмотры (по возрасту: щенки - 3 мес, взрослые - 1 год, пожилые - 6 мес)
- Контроль веса (для пород, склонных к ожирению)

## ❓ FAQ

**Q: Почему breed_standards.json не нужен для персонализации?**  
A: Стандарты организаций (CFA, AKC) нужны только для энциклопедии пород, не для рекомендаций товаров/курсов.

**Q: Можно ли работать без `nutrition_params` в товарах?**  
A: Нет, это критически важно. Без БЖУ и ингредиентов невозможно подобрать корм по здоровью и аллергиям.

**Q: Сколько времени займет заполнение всех товаров?**  
A: Для топ-50 товаров - 1 день вручную. Для всех товаров - 2-3 недели (парсинг + ручная проверка).

**Q: Что делать с товарами не-корм (амуниция, игрушки)?**  
A: Для них не нужен `nutrition_params`, но нужен `product_params` (размер, материал, цвет).
