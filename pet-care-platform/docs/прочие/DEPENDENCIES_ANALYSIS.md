**Текущая версия**: 4.2.8  
**Целевая версия**: 5.1.5 (LTS)  
**Приоритет**: 🔴 Высокий

**Изменения в Django 5.x**:
- Python 3.10+ обязателен
- Изменения в работе с JSON полями
- Новые возможности в ORM
- Улучшения производительности
- Изменения в middleware

**Потенциальные проблемы**:
- Изменения в `django.utils.translation`
- Изменения в работе с JSON полями
- Изменения в middleware
- Изменения в валидации форм

**План миграции**:
1. Обновить Python до 3.10+ (если необходимо)
2. Обновить Django до 5.0.x сначала
3. Протестировать все функции
4. Обновить до 5.1.5
5. Исправить breaking changes

**Время**: 3-5 дней
