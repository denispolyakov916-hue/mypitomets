# 🚀 Питомец+ - Быстрый запуск

## 📋 Системные требования

- **Python 3.10+** (для бэкенда)
- **Node.js 18+** (для фронтенда)
- **PostgreSQL** или **SQLite** (база данных)

## 🛠️ Быстрый запуск

### Шаг 1: Запуск бэкенда
```batch
REM Откройте командную строку в корневой папке проекта
start-backend.bat
```
Бэкенд запустится на `http://localhost:8077`

### Шаг 2: Запуск фронтенда
```batch
REM Откройте новую командную строку в корневой папке проекта
start-frontend.bat
```
Фронтенд запустится на `http://localhost:5199`

## 📦 Что делают скрипты запуска

### start-backend.ps1
1. ✅ Активирует виртуальное окружение Python
2. ✅ Проверяет и применяет миграции базы данных
3. ✅ Загружает тестовые данные и курсы
4. ✅ Запускает Django сервер на порту 8001

### start-frontend.ps1
1. ✅ Устанавливает переменную окружения `VITE_API_URL`
2. ✅ Устанавливает зависимости npm (при первом запуске)
3. ✅ Запускает Vite dev server на порту 5174 с прокси на бэкенд

## 🐛 Устранение неполадок

### Бэкенд не запускается
1. Проверьте, что Python установлен: `python --version`
2. Проверьте виртуальное окружение: `backend\venv\Scripts\python.exe --version`
3. Примените миграции вручную: `python manage.py migrate`

### Фронтенд не запускается
1. Проверьте Node.js: `node --version`
2. Очистите кэш: `rm -rf node_modules && npm install`
3. Проверьте порт: убедитесь, что 5199 свободен

### API не отвечает
1. Проверьте, что бэкенд запущен на порту 8077
2. Проверьте переменную окружения VITE_API_URL
3. Проверьте логи бэкенда на ошибки
